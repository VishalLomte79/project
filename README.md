# Restaurantly App

A responsive mobile-first restaurant application built with React Native, Expo, and React Native Web.

## Features

- **Responsive Design**: Works seamlessly on Mobile (iOS/Android) and Web.
- **Home Screen**: Hero section with a single-page scrollable layout including Menu, Chefs, and Contact sections.
- **Menu**: Browse categories and view delicious dishes.
- **Dish Details**: View detailed information and watch "making of" videos for dishes.
- **Chefs**: Meet our professional chefs.
- **Contact**: Location, hours, and contact information.
- **Dark Theme**: Elegant dark UI with gold accents.

## Tech Stack

- **Framework**: React Native (Expo)
- **Language**: TypeScript
- **Navigation**: React Navigation (Bottom Tabs + Native Stack)
- **Web Support**: React Native Web
- **Video**: expo-av

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/VishalLomte79/project.git
    cd project
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the app**
    ```bash
    npx expo start
    ```

4.  **Open on your device**
    -   **Mobile**: Scan the QR code with the Expo Go app.
    -   **Web**: Press `w` in the terminal to open in the browser.

## Project Structure

-   `src/screens`: Application screens (Home, Menu, etc.)
-   `src/components`: Reusable UI components.
-   `src/navigation`: Navigation configuration.
-   `src/api`: Mock API and types.
-   `src/theme`: Theme constants (colors, typography, spacing).
